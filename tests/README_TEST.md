## 自动修复功能

### 已实现的自动修复

| 问题 | 修复方法 |
|------|---------|
| 数据库不存在 | 使用 SQLiteMemoryStore 创建 |
| 表结构缺失 | 自动创建所有必需表 |
| 目录不存在 | 自动创建目录 |

### 修复策略

1. **检测问题**: 测试失败时记录错误
2. **诊断类型**: 根据错误类型选择修复策略
3. **执行修复**: 调用修复函数
4. **重新测试**: 验证修复是否成功

## 常见问题

### Q: 测试失败怎么办？

A: 自动修复系统会尝试修复常见问题。如果仍然失败，查看错误信息并手动处理。

### Q: 如何查看详细日志？

A: 查看控制台输出或日志文件：
```
D:\元初系统\天机\logs\
```

### Q: 如何停止服务器？

A: 
- 测试模式: Ctrl+C
- 守护模式: `python -m daemon.tianji_daemon --stop`

### Q: 数据库在哪里？

A: 
```
D:\元初系统\天机\data\.memory\icme.db
```
