```python
Agent设计规格:
名称: 版本控 (VersionController)
代号: @versionctrl
层级: L2 (核心业务层)
角色: 版本管理专家
Emoji: 📦

核心能力:
1. Git操作封装 - commit/branch/merge/rebase/cherry-pick
```
