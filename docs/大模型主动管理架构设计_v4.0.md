## 一、核心革命：从被动存储到主动认知

### 1.1 架构演进对比

#### 当前架构 (v3.1) - 被动存储模式

```
用户输入 → Trae IDE → 记忆系统API → 存储到SQLite
         ↓
         被动检索 (用户明确要求时)
```

**问题**:
- ❌ 记忆系统是被动的"仓库", 不参与认知过程
- ❌ 大模型无法主动决定存储/检索策略
- ❌ 跨平台AI无法共享记忆上下文
- ❌ 记忆价值评估依赖简单规则, 缺乏智能

#### 新架构 (v4.0) - 主动认知模式

```
用户输入 → Trae IDE ↔ 大模型(DeepSeek/GPT-4) ↔ 记忆系统
                    ↓                           ↓
                主动决策                      主动管理
                ├─ 是否存储?                  ├─ 价值评分
                ├─ 存储到哪层?                ├─ 关联分析
                ├─ 是否检索?                  ├─ 遗忘曲线
                └─ 检索什么?                  └─ 知识融合
```

**优势**:
- ✅ 大模型成为记忆系统的"大脑"
- ✅ 主动决策存储/检索策略
- ✅ 跨平台AI共享统一记忆上下文
- ✅ 智能价值评估和知识提取

#### 2.2.1 LLM Integration Layer (大模型集成层)

**职责**: 统一管理多个大模型API, 提供智能路由和负载均衡

**核心类设计**:

```python
class LLMIntegrationLayer:
    """大模型集成层 - 统一管理多个LLM API"""
    
    def __init__(self, config: LLMConfig):
        self.adapters: Dict[str, LLMAdapter] = {
            "deepseek": DeepSeekAdapter(config.deepseek_api_key),
            "gpt4": GPT4Adapter(config.openai_api_key),
            "claude": ClaudeAdapter(config.anthropic_api_key),
        }
        self.router = LLMRouter()
        self.cache = ResponseCache()
        
    async def analyze_memory_value(self, content: str, context: Dict) -> float:
        """让大模型评估记忆价值 (0.0-1.0)"""
        prompt = f"""
        请评估以下内容的记忆价值 (0.0-1.0):
        
        内容: {content}
        上下文: {json.dumps(context, ensure_ascii=False)}
        
        评估维度:
        1. 重要性 (是否包含关键决策/知识)
        2. 唯一性 (是否重复已有记忆)
        3. 时效性 (是否长期有效)
        4. 关联性 (是否与其他记忆关联)
        
        返回JSON: {{"value_score": 0.85, "reason": "..."}}
        """
        
        response = await self.router.route(prompt, task="value_assessment")
        return response["value_score"]
    
    async def decide_storage_strategy(self, content: str, context: Dict) -> Dict:
        """让大模型决定存储策略"""
        prompt = f"""
        请决定以下内容的存储策略:
        
        内容: {content}
        当前会话上下文: {json.dumps(context, ensure_ascii=False)}
        
        决策:
        1. 是否存储? (true/false)
        2. 存储到哪层? (sensory/working/short_term/episodic/semantic/meta)
        3. 标签建议 (["tag1", "tag2"])
        4. 优先级
        5. 关联记忆ID (如果有)
        
        返回JSON格式。
        """
        
        return await self.router.route(prompt, task="storage_decision")
    
    async def extract_knowledge(self, content: str) -> List[Dict]:
        """让大模型从内容中提取知识三元组"""
        prompt = f"""
        请从以下内容中提取知识三元组 (主体, 关系, 客体):
        
        内容: {content}
        
        示例:
        "Python是编程语言" → ("Python", "is_a", "编程语言")
        "FastAPI用于构建REST API" → ("FastAPI", "used_for", "REST API")
        
        返回JSON数组: [{{"subject": "...", "relation": "...", "object": "..."}}]
        """
        
        return await self.router.route(prompt, task="knowledge_extraction")
```

```python
from abc import ABC, abstractmethod
from typing import Dict, Any, Optional
import httpx

class LLMAdapter(ABC):
    """大模型适配器基类"""
    
    @abstractmethod
    async def call(self, prompt: str, **kwargs) -> Dict[str, Any]:
        """调用大模型API"""
        pass
    
    @abstractmethod
    async def stream_call(self, prompt: str, **kwargs):
        """流式调用大模型API"""
        pass

class DeepSeekAdapter(LLMAdapter):
    """DeepSeek大模型适配器"""
    
    def __init__(self, api_key: str, base_url: str = "https://api.deepseek.com/v1"):
        self.api_key = api_key
        self.base_url = base_url
        self.client = httpx.AsyncClient(timeout=60.0)
        
    async def call(self, prompt: str, model: str = "deepseek-chat", **kwargs) -> Dict[str, Any]:
        """调用DeepSeek API"""
        headers = {
            "Authorization": f"Bearer {self.api_key}",
            "Content-Type": "application/json"
        }
        
        payload = {
            "model": model,
            "messages": [
                {"role": "system", "content": "你是AI记忆系统的认知核心, 负责主动管理记忆。"},
                {"role": "user", "content": prompt}
            ],
            "temperature": kwargs.get("temperature", 0.3),
            "max_tokens": kwargs.get("max_tokens", 2000),
            "response_format": {"type": "json_object"}
        }
        
        response = await self.client.post(
            f"{self.base_url}/chat/completions",
            headers=headers,
            json=payload
        )
        
        result = response.json()
        return json.loads(result["choices"][0]["message"]["content"])
```

**智能路由器**:

```python
class LLMRouter:
    """大模型智能路由器 - 根据任务类型选择最优模型"""
    
    def __init__(self):
        self.task_model_mapping = {
            "value_assessment": "deepseek",      # DeepSeek擅长价值评估
            "storage_decision": "deepseek",      # DeepSeek擅长决策
            "knowledge_extraction": "gpt4",      # GPT-4擅长知识提取
            "summarization": "claude",           # Claude擅长总结
            "creative_writing": "gpt4",          # GPT-4擅长创作
        }
        self.load_balancer = LoadBalancer()
        
    async def route(self, prompt: str, task: str, **kwargs) -> Dict[str, Any]:
        """智能路由到最优模型"""
        preferred_model = self.task_model_mapping.get(task, "deepseek")
        
        # 检查负载和可用性
        if not self.load_balancer.is_available(preferred_model):
            preferred_model = self.load_balancer.get_fallback(task)
        
        adapter = self.adapters[preferred_model]
        self.load_balancer.record_request(preferred_model)
        
        return await adapter.call(prompt, **kwargs)
```

#### 2.2.2 Active Memory Protocol (主动记忆协议层)

**职责**: 定义大模型与记忆系统的交互协议

**核心协议**:

```python
from dataclasses import dataclass
from typing import List, Dict, Optional
from enum import Enum

class MemoryAction(Enum):
    STORE = "store"
    RETRIEVE = "retrieve"
    UPDATE = "update"
    DELETE = "delete"
    CONSOLIDATE = "consolidate"
    FORGET = "forget"

@dataclass
class MemoryDecision:
    """大模型的记忆决策"""
    action: MemoryAction
    confidence: float  # 0.0-1.0
    reasoning: str
    target_layer: Optional[str] = None
    tags: Optional[List[str]] = None
    priority: Optional[str] = None
    related_memories: Optional[List[str]] = None
    value_score: Optional[float] = None

@dataclass
class RetrievalStrategy:
    """大模型的检索策略"""
    query_expansion: List[str]  # 扩展查询词
    semantic_search: bool       # 是否使用语义搜索
    layer_filter: List[str]     # 检索哪些层
    time_range: Optional[tuple] # 时间范围
    min_relevance: float        # 最小相关度
    max_results: int            # 最大结果数

class ActiveMemoryProtocol:
    """主动记忆协议 - 大模型与记忆系统的交互协议"""
    
    async def process_user_input(
        self,
        user_input: str,
        context: Dict,
        llm_layer: LLMIntegrationLayer
    ) -> MemoryDecision:
        """处理用户输入, 让大模型决定记忆策略"""
        
        # 1. 让大模型分析输入价值
        value_score = await llm_layer.analyze_memory_value(user_input, context)
        
        # 2. 让大模型决定存储策略
        storage_decision = await llm_layer.decide_storage_strategy(user_input, context)
        
        # 3. 构建记忆决策
        decision = MemoryDecision(
            action=MemoryAction.STORE if storage_decision["should_store"] else MemoryAction.RETRIEVE,
            confidence=storage_decision["confidence"],
            reasoning=storage_decision["reasoning"],
            target_layer=storage_decision["layer"],
            tags=storage_decision["tags"],
            priority=storage_decision["priority"],
            value_score=value_score
        )
        
        return decision
    
    async def build_retrieval_strategy(
        self,
        query: str,
        context: Dict,
        llm_layer: LLMIntegrationLayer
    ) -> RetrievalStrategy:
        """让大模型构建检索策略"""
        
        prompt = f"""
        用户查询: {query}
        当前上下文: {json.dumps(context, ensure_ascii=False)}
        
        请构建最优检索策略:
        1. 查询扩展 (同义词/相关词)
        2. 是否使用语义搜索?
        3. 检索哪些记忆层?
        4. 时间范围限制?
        5. 最小相关度阈值?
        6. 最大结果数?
        
        返回JSON格式。
        """
        
        strategy_dict = await llm_layer.router.route(prompt, task="retrieval_strategy")
        
        return RetrievalStrategy(
            query_expansion=strategy_dict["query_expansion"],
            semantic_search=strategy_dict["semantic_search"],
            layer_filter=strategy_dict["layer_filter"],
            time_range=tuple(strategy_dict["time_range"]) if strategy_dict.get("time_range") else None,
            min_relevance=strategy_dict["min_relevance"],
            max_results=strategy_dict["max_results"]
        )
```

#### 2.2.3 AI Platform Adapters (AI平台适配层)

**职责**: 让记忆系统主动适配不同AI平台的对话过程

**核心设计**:

```python
from abc import ABC, abstractmethod
from typing import Dict, Any, Optional, Callable
import asyncio

class AIPlatformAdapter(ABC):
    """AI平台适配器基类"""
    
    @abstractmethod
    async def intercept_user_input(self, user_input: str, context: Dict) -> str:
        """拦截用户输入, 注入记忆上下文"""
        pass
    
    @abstractmethod
    async def intercept_ai_response(self, ai_response: str, context: Dict) -> str:
        """拦截AI响应, 触发记忆存储"""
        pass
    
    @abstractmethod
    async def get_conversation_context(self) -> Dict:
        """获取当前对话上下文"""
        pass

class TraeAdapter(AIPlatformAdapter):
    """Trae IDE平台适配器"""
    
    def __init__(self, memory_system: "ActiveMemorySystem", llm_layer: LLMIntegrationLayer):
        self.memory_system = memory_system
        self.llm_layer = llm_layer
        self.session_id = None
        self.conversation_history = []
        
    async def intercept_user_input(self, user_input: str, context: Dict) -> str:
        """拦截用户输入, 主动检索相关记忆"""
        
        # 1. 让大模型构建检索策略
        retrieval_strategy = await self.memory_system.protocol.build_retrieval_strategy(
            query=user_input,
            context=context,
            llm_layer=self.llm_layer
        )
        
        # 2. 执行检索
        relevant_memories = await self.memory_system.retrieve_with_strategy(
            query=user_input,
            strategy=retrieval_strategy
        )
        
        # 3. 构建记忆上下文
        memory_context = self._build_memory_context(relevant_memories)
        
        # 4. 注入到用户输入
        enhanced_input = f"""
[记忆上下文 - 自动注入]
{memory_context}

[用户输入]
{user_input}
"""
        
        # 5. 记录对话历史
        self.conversation_history.append({
            "role": "user",
            "content": user_input,
            "timestamp": time.time()
        })
        
        return enhanced_input
    
    async def intercept_ai_response(self, ai_response: str, context: Dict) -> str:
        """拦截AI响应, 主动存储重要记忆"""
        
        # 1. 让大模型评估响应价值
        decision = await self.memory_system.protocol.process_user_input(
            user_input=ai_response,
            context={**context, "conversation_history": self.conversation_history},
            llm_layer=self.llm_layer
        )
        
        # 2. 根据决策存储记忆
        if decision.action == MemoryAction.STORE and decision.confidence > 0.7:
            memory_id = await self.memory_system.store(
                content=ai_response,
                layer=decision.target_layer,
                tags=decision.tags,
                priority=decision.priority,
                metadata={
                    "source": "trae_ai_response",
                    "session_id": self.session_id,
                    "value_score": decision.value_score,
                    "reasoning": decision.reasoning
                }
            )
            
            # 3. 提取知识并更新图谱
            knowledge_triples = await self.llm_layer.extract_knowledge(ai_response)
            await self.memory_system.update_knowledge_graph(knowledge_triples)
        
        # 4. 记录对话历史
        self.conversation_history.append({
            "role": "assistant",
            "content": ai_response,
            "timestamp": time.time()
        })
        
        return ai_response
    
    def _build_memory_context(self, memories: List[Dict]) -> str:
        """构建记忆上下文字符串"""
        if not memories:
            return "无相关记忆"
        
        context_lines = []
        for i, mem in enumerate(memories[:5], 1):  # 最多5条
            context_lines.append(
                f"{i}. [{mem['layer']}] {mem['content'][:100]}... "
                f"(相关度: {mem['relevance']:.2f})"
            )
        
        return "\n".join(context_lines)

class VSCodeAdapter(AIPlatformAdapter):
    """VSCode平台适配器 (类似Trae)"""
    pass

class CursorAdapter(AIPlatformAdapter):
    """Cursor平台适配器"""
    pass
```

#### 2.2.4 Persistent Service Layer (持久化服务层)

**职责**: 实现记忆系统持久化激活, 跨Trae实例保持状态

**Windows服务实现**:

```python
import win32serviceutil
import win32service
import win32event
import servicemanager
import asyncio
from threading import Thread

class AIMemorySystemService(win32serviceutil.ServiceFramework):
    """AI记忆系统Windows服务"""
    
    _svc_name_ = "AIMemorySystem"
    _svc_display_name_ = "AI Memory System - Cognitive Core"
    _svc_description_ = "大模型主动管理的AI记忆系统, 跨平台持久化服务"
    
    def __init__(self, args):
        win32serviceutil.ServiceFramework.__init__(self, args)
        self.hWaitStop = win32event.CreateEvent(None, 0, 0, None)
        self.loop = None
        self.server = None
        
    def SvcStop(self):
        """停止服务"""
        self.ReportServiceStatus(win32service.SERVICE_STOP_PENDING)
        if self.loop:
            self.loop.call_soon_threadsafe(self.loop.stop)
        win32event.SetEvent(self.hWaitStop)
        
    def SvcDoRun(self):
        """运行服务"""
        servicemanager.LogMsg(
            servicemanager.EVENTLOG_INFORMATION_TYPE,
            servicemanager.PYS_SERVICE_STARTED,
            (self._svc_name_, '')
        )
        
        # 启动异步事件循环
        self.loop = asyncio.new_event_loop()
        asyncio.set_event_loop(self.loop)
        
        # 启动FastAPI服务器
        from server.main import app
        import uvicorn
        
        config = uvicorn.Config(
            app,
            host="0.0.0.0",
            port=8765,
            loop="asyncio",
            log_level="info"
        )
        self.server = uvicorn.Server(config)
        
        # 在新线程中运行服务器
        server_thread = Thread(target=self.loop.run_until_complete, args=(self.server.serve(),))
        server_thread.start()
        
        # 等待停止信号
        win32event.WaitForSingleObject(self.hWaitStop, win32event.INFINITE)
        server_thread.join()

if __name__ == '__main__':
    win32serviceutil.HandleCommandLine(AIMemorySystemService)
```

## 三、核心工作流程

### 3.1 主动记忆存储流程

```
用户输入 → Trae IDE
         ↓
    TraeAdapter.intercept_user_input()
         ↓
    LLM分析价值 + 决定存储策略
         ↓
    ┌─────────────────┐
    │ value_score > 0.7? │
    └─────────────────┘
         ↓ 是                ↓ 否
    存储到指定层         仅记录到sensory层
         ↓
    提取知识三元组
         ↓
    更新知识图谱
         ↓
    返回增强输入给Trae
```

### 3.2 主动记忆检索流程

```
用户查询 → Trae IDE
         ↓
    TraeAdapter.intercept_user_input()
         ↓
    LLM构建检索策略
         ↓
    ┌──────────────────────┐
    │ query_expansion      │
    │ semantic_search      │
    │ layer_filter         │
    │ time_range           │
    └──────────────────────┘
         ↓
    执行多维度检索
         ↓
    ┌──────────────────────┐
    │ FTS5关键词搜索        │
    │ 向量语义搜索          │
    │ 知识图谱关联查询      │
    └──────────────────────┘
         ↓
    结果融合与排序
         ↓
    构建记忆上下文
         ↓
    注入到用户输入
         ↓
    返回给Trae
```

### 3.3 跨平台AI对话适配流程

```
用户在Trae中对话
         ↓
    TraeAdapter拦截输入/输出
         ↓
    记忆系统主动管理记忆
         ↓
    存储到共享记忆库
         ↓
用户在VSCode中继续对话
         ↓
    VSCodeAdapter拦截输入
         ↓
    从共享记忆库检索Trae会话记忆
         ↓
    注入上下文, 实现跨平台连续对话
```

## 七、性能优化

### 7.1 大模型调用优化

1. **批量处理**: 合并多个记忆评估请求
2. **缓存机制**: 缓存常见查询的决策结果
3. **异步调用**: 使用asyncio并发调用多个大模型
4. **降级策略**: 大模型不可用时降级到规则引擎

### 7.2 记忆检索优化

1. **预加载**: 启动时预加载热点记忆到缓存
2. **索引优化**: 定期重建FTS5和向量索引
3. **分区检索**: 按时间/标签分区, 减少检索范围
4. **结果缓存**: 缓存检索结果, 避免重复计算
