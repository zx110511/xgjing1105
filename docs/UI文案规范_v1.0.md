## 📋 文案原则

### 核心原则

1. **用户中心**: 以用户视角编写，使用"您"而非"用户"
2. **简洁明了**: 避免冗长，直击要点，每句不超过20字
3. **积极正面**: 强调成功而非失败，使用积极词汇
4. **一致性**: 统一术语和表达方式，避免歧义
5. **专业性**: 保持专业但不晦涩，通俗易懂

### 文案风格

- **语气**: 友好、专业、简洁
- **人称**: 第二人称"您"
- **时态**: 现在时为主
- **标点**: 中文标点，句末不加句号（按钮除外）
