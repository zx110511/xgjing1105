## 三、测试执行流程

### Phase 1: 一键安装测试（模拟新用户）

```powershell
# 1. 清理环境
Remove-Item -Recurse -Force "C:\Program Files\天机v9.1" -ErrorAction SilentlyContinue
Remove-Item -Recurse -Force "$env:APPDATA\天机" -ErrorAction SilentlyContinue

# 2. 模拟双击安装
Start-Process "天机v9.1-Setup.exe" -ArgumentList "/SILENT" -Wait

# 3. 验证安装结果
Test-Installation

# 4. 首次启动测试
Start-Process "C:\Program Files\天机v9.1\天机v9.1.exe"
Start-Sleep 10
Test-FirstLaunch
```

### Phase 2: AI平台适配测试

```powershell
# 启动天机后端
Start-TianjiBackend

# 测试Trae IDE适配
Test-TraeIDEAdapter

# 测试Cursor适配
Test-CursorAdapter

# 测试Claude适配
Test-ClaudeAdapter
```

### Phase 3: 性能测试

```powershell
# 冷启动测试
Measure-ColdStart

# 内存测试
Measure-MemoryUsage

# API压力测试
Invoke-APIStressTest
```
