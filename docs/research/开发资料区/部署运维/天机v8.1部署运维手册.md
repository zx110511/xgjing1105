# 天机v9.1 部署运维手册

> 数据根基: deploy/ + scripts/ + tianji_launcher.py + tianji_service.py

---

## 快速启动

### Windows桌面版（推荐）
```batch
# 双击 tray_launcher 或 运行
python tianji_launcher.py
```

### 命令行版
```batch
# 方式1: 直接启动服务
python server/main.py

# 方式2: uvicorn启动
uvicorn server.main:app --host 0.0.0.0 --port 8770

# 方式3: 使用脚本
scripts\start.bat
```

## 停止服务

```batch
scripts\stop.bat
scripts\kill.bat          # 强制终止
```

## 重启服务

```batch
scripts\restart.bat
scripts\quick_restart.bat  # 快速重启
scripts\force_restart.bat  # 强制重启（杀端口+重建）
```

# PyInstaller打包

## 运维常用操作

### 查看服务状态
```batch
scripts\status.bat
```

### 查看进程
```powershell
netstat -ano | findstr 8770
```

### 清除端口占用
```powershell
# 自动由 launcher 处理
# 或手动: taskkill /F /PID <PID>
```

### 查看日志
```
logs/launcher.log       # 启动器日志
logs/governance.log     # 治理系统日志
logs/service_stdout.log # 服务标准输出
logs/service_stderr.log # 服务错误输出
```

### 查看审计报告
```
logs/audit_reports/governance_audit_*.json
```

### 守护进程
```batch
scripts\daemon_start.bat   # 启动守护进程
```

### 命令行工具
```batch
scripts\cli.bat
scripts\cli.ps1
```

## 开机自启

由 `tianji_service.py` 负责注册Windows计划任务实现开机自启动

## 故障排查

1. **端口冲突**: 检查 8770 端口是否被占用
2. **Python路径**: 确保 Python SDK 在 `C:\Users\Administrator\python-sdk\python3.12.9\`
3. **Web前端未构建**: 运行 `cd web && npm install && npm run build`
4. **数据库锁定**: 检查 `data/tianji.db` 文件权限
5. **治理组件未启用**: 检查 `logs/governance.log` 排查导入错误
