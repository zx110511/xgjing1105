## 一、系统级端点（/）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | / | 根路径，返回服务信息（JSON）或SPA首页 |
| GET | /dashboard | SPA路由 → 仪表盘页面 |
| GET | /memory | SPA路由 → 记忆管理页面 |
| GET | /knowledge-graph | SPA路由 → 知识图谱页面 |
| GET | /config | SPA路由 → 系统配置页面 |
| GET | /monitor | SPA路由 → 监控日志页面 |
| GET | /api/health | 健康检查 → HealthStatus JSON |
| GET | /api/stats | 记忆统计 → MemoryStats JSON |
| GET | /api/system/stats | 系统三维统计数据（33模块实时+累计+历史） |
| GET | /api/shutdown | 优雅关闭服务 |
