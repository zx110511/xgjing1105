# 天机v9.1 配置说明

> 数据根基: server/main.py + tianji_service.py + config/* + deploy/

---

## 环境变量

| 变量 | 默认值 | 说明 |
|------|--------|------|
| AI_MEMORY_ROOT | 项目根目录 | 数据存储根目录 |
| AI_MEMORY_PORT | 8770 | FastAPI服务端口 |
| AI_MEMORY_HOST | 0.0.0.0 | 绑定地址 |
| TIANJI_EDITION | source-v9.1 | 版本标识 |

## 版本标识映射

| TIANJI_EDITION | 标签 |
|----------------|------|
| compiled-exe | 编译版 (天机v9.1.exe) |
| source-v9.1 | 源码版 v8.0 (平台化研究) |
| source-v9.1 | 源码版 v9.1 (统一架构) |

## 关键路径（运行时自动创建）

```
{AI_MEMORY_ROOT}/
├── logs/               # 日志目录
│   ├── launcher.log     # 启动器日志
│   ├── governance.log   # 治理日志
│   ├── service_stdout.log # 服务stdout
│   ├── service_stderr.log # 服务stderr
│   └── audit_reports/   # 审计报告JSON
├── data/               # 数据目录
│   └── tianji.db       # SQLite记忆数据库
├── backup/             # 备份目录
└── daemon/             # 守护进程
    └── tianji.pid      # 进程PID文件
```

## 服务启动参数

```bash
# 源码版
python server/main.py
uvicorn server.main:app --host 0.0.0.0 --port 8770

# 编译版
天机v9.1.exe
```
