# 天机v9.1 桌面版验收标准

### 1. 安装包基础验证
- [ ] 安装包大小 < 100MB
- [ ] 安装过程无错误/警告
- [ ] 安装后程序可正常启动

- [ ] Python后端自动启动（端口8778）
- [ ] 前端界面正常渲染

### 3. 桌面集成验证
- [ ] 系统托盘图标显示
