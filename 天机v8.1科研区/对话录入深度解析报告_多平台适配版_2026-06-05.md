#### 层级2: API处理层 (active_routes.py)

**三层写入操作**:

| 写入顺序 | 层级 | 存储内容 | 标签 | 元数据 |
|----------|------|----------|------|--------|
| **1** | L0 Sensory | 用户输入全文 | auto-capture, user-input, agent:{id} | source="capture_conversation", turn_id, platform, role="user" |
| **2** | L1 Working | AI回复前2000字符 | auto-capture, ai-response, agent:{id} | source="capture_conversation", turn_id, platform, role="assistant", full_length |
| **3** | L3 Episodic | 结构化对话事件 | auto-capture, conversation-event, agent:{id} | source="capture_conversation", turn_id, 各项长度统计 |

```python
turn_id = hashlib.md5(
```

#### 层级3: 引擎层 (hybrid_engine.py)

**remember()核心流程**:

```python
def remember(self, content, layer, tags, metadata):
    # 1. TCL归一化
    _, canonical_ids = self._tcl_normalizer.normalize_content(content)
    metadata["tcl_canonical_ids"] = canonical_ids

    # 2. QualityGate检查
    gate_result = self._quality_gate.check(content, layer, tags, priority, existing)
    actual_layer = gate_result.target_layer

    # 3. 生成entry_id
    entry_id = hashlib.sha256(
        f"{content}{time.time()}{uuid.uuid4()}".encode()
    ).hexdigest()[:16]

    # 4. 构造entry_dict
    entry_dict = {
        "id": entry_id,
        "content": content,
        "layer": actual_layer,
        "tags": tags,
        "priority": priority,
        "value_score": self._calc_value_score(priority),
        "access_count": 0,
        "created_at": time.time(),
        "last_accessed": time.time(),
        "size_bytes": len(content.encode("utf-8")),
        "metadata": metadata,
        "related_ids": [],
        "changelog": [],
    }

    # 5. SQLite存储
    self._store.insert(entry_dict)

    # 6. AssetRegistry注册 (策略D)
    content_hash = self._asset_registry.compute_content_hash(content)
    atom = AssetAtom(memory_id=entry_id, layer=actual_layer, ...)
    asset_id = self._asset_registry.register(atom, content=content)

    return {"id": entry_id, "status": "stored", "asset_id": asset_id}
```

#### 层级4: 存储层 (sqlite_store.py)

**insert()操作细节**:

```python
def insert(self, entry):
    conn = self._get_conn()

    # 1. 中文分词
    segmented = tokenize_for_fts(entry["content"])

    # 2. SQL插入
    conn.execute("""
        INSERT INTO memories (
            id, content, content_segmented, layer, tags, priority, value_score,
            access_count, created_at, last_accessed, size_bytes,
            metadata, related_ids, changelog
        ) VALUES (?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?, ?)
    """, (
        entry["id"],
        entry["content"],
        segmented,  # 分词后的内容
        entry["layer"],
        json.dumps(entry.get("tags", [])),  # JSON序列化
        entry.get("priority", "medium"),
        entry.get("value_score", 0.5),
        entry.get("access_count", 0),
        entry.get("created_at", time.time()),
        entry.get("last_accessed", time.time()),
        len(entry["content"].encode("utf-8")),
        json.dumps(entry.get("metadata", {})),  # JSON序列化
        json.dumps(entry.get("related_ids", [])),
        json.dumps(entry.get("changelog", [])),
    ))

    # 3. 提交事务
    conn.commit()
```

## 三、多平台适配机制

### 3.1 平台识别与路由

**平台标识传递链**:
```
IDE对话结束 → on_conversation_end(platform="trae"/"qoder")
    ↓
ConversationHookManager.trigger_by_platform(platform)
    ↓
TraeConversationHook / QoderConversationHook
    ↓
HTTP POST payload["platform"] = "trae"/"qoder"
    ↓
capture_conversation(request.platform)
    ↓
metadata["platform"] = "trae"/"qoder"
    ↓
SQLite存储: memories.metadata = {"platform": "trae"/"qoder", ...}
```

### 3.2 平台差异处理

**当前实现**: 平台标识仅用于元数据记录，不影响存储逻辑

**潜在扩展点**:

| 扩展方向 | Trae平台 | Qoder平台 | 实现方式 |
|----------|----------|-----------|----------|
| **存储策略** | 可配置不同层级策略 | 可配置不同层级策略 | platform_config.json |
| **标签前缀** | trae:tag | qoder:tag | tags = [f"{platform}:{tag}" for tag in tags] |
| **Agent映射** | trae → @lingxi | qoder → @tianshu | agent_mapping.json |
| **会话隔离** | trae_session_* | qoder_session_* | conversation_id = f"{platform}_{session_id}" |

```python
# 添加到server/main.py的startup_event
import asyncio

async def auto_capture_conversations():
    """定时扫描并捕获对话"""
    while True:
        try:
            # 扫描Trae/Qoder的对话日志
            conversations = scan_conversation_logs()

            for conv in conversations:
                if not is_captured(conv["turn_id"]):
                    # 调用capture_conversation
                    await capture_conversation(conv)

            await asyncio.sleep(60)  # 每分钟扫描一次
        except Exception as e:
            logger.error(f"自动捕获失败: {e}")

# 启动时添加任务
asyncio.create_task(auto_capture_conversations())
```
